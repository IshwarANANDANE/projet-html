Un site Web statique est un site où chacune des pages est créée en HTML.
Un ordinateur qui se connecte au serveur, demande une page. Celle-ci lui est directement servie (elle est stockée dans sa version finale sur le serveur).

Un site Web dynamique est un site Web dont les pages sont générées dynamiquement à la demande. Le contenu est obtenu (par exemple)
en combinant l'utilisation d'un langage de scripts ou de programmation et une base de données. Il s'agit souvent de JavaScript ou de PHP
pour le langage et de MySQL pour la base de données.

  • Balises de titre : de h1 à h6 en taille de plus grande en plus petite
  • Paragraphe : balise p
  • Les balises span et div permettent de regrouper tout type de contenu HTML pour lui appliquer des propriétés
  • Un lien peut pointer sur tout type d’objet informatique, pas uniquement sur une page web, balise ‘a’
